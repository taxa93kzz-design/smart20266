# Информатика БЖБ онлайн тест

Бұл репозиторийде оқушы сайттың өзінде орындайтын Информатика БЖБ онлайн тесті орналасқан.

## Файл

- `index.html` - негізгі сайт файлы.

## GitHub Pages

Сайтты онлайн ашу үшін репозиторий баптауынан GitHub Pages қосыңыз:

1. Repository Settings бөліміне кіріңіз.
2. Pages бөлімін ашыңыз.
3. Source ретінде `Deploy from a branch` таңдаңыз.
4. Branch: `main`, Folder: `/root` таңдаңыз.
5. Save басыңыз.
